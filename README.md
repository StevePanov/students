# Тестовое задание DoubleTapp
Как запустить:

бэк:
 - переходим в папку  `cd server`
 - устанавливаем библиотеки  `npm i`
 - запускаем  `mongod`
 - запускаем сервер  `npm run dev`

фронт:
 - выходим из папки server  `cd ..`
 - переходим в папку  `cd client`
 - устанавливаем библиотеки  `npm i`
 - запускаем фронт  `npm run start`
